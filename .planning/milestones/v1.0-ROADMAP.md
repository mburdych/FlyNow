# Milestone v1.0: milestone

**Status:** SHIPPED 2026-04-24
**Phases:** 1-4
**Total Plans:** 8

## Overview

FlyNow v1.0 delivered the full MVP scope: weather analysis, time-window evaluation, Home Assistant integration, GO notifications, Lovelace card UX, flight logging, and multi-site planning support while preserving backward compatibility for existing automations.

## Phases

### Phase 1: Core Integration

**Goal:** Home Assistant fetches, analyzes, and exposes balloon-flyable condition data via sensors and config flow.
**Plans:** 3/3 complete

- 01-01: Weather client + analyzer baseline
- 01-02: Time-window and config flow setup
- 01-03: Coordinator and sensor runtime integration

### Phase 2: Notifications & Card

**Goal:** Crew and pilot receive good-window notifications and have a dashboard card for operational status.
**Plans:** 2/2 complete

- 02-01: Notification orchestration and dedup
- 02-02: Lovelace card with two-window-first rendering

### Phase 3: Flight Logging

**Goal:** Crew logs completed flights through the card and stores data locally for future learning.
**Plans:** 2/2 complete

- 03-01: Backend services + atomic JSON persistence
- 03-02: Card form and history integration

### Phase 4: Multi-site forecast planning card

**Goal:** Compare fixed launch sites in one planning-first card while keeping `binary_sensor.flynow_status` backward-compatible.
**Plans:** 1/1 complete

- 04-01: Static 3-site projection and comparison-first card

## Milestone Summary

**Key outcomes:**
- End-to-end HAOS operation stabilized in production.
- `flynow.log_flight` and `flynow.list_flights` implemented with atomic persistence.
- Multi-site planning output available for `lzmada`, `katarinka`, `nitra-luka`.
- EASA/SERA civil-twilight day limits enforced and surfaced in UI.

**Issues resolved during milestone:**
- Open-Meteo contract fix (`ceiling` -> `cloud_base`)
- Analyzer `None`-value crash hardening
- Missing `binary_sensor.py` platform registration
- Card config/runtime contract fixes (`setConfig`, `getCardSize`, key mapping)
