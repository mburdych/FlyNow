## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label micro gaps, inline status chip spacing |
| sm | 8px | Row spacing inside condition blocks |
| md | 16px | Default internal card padding and tile spacing |
| lg | 24px | Separation between top summary tiles and detail section |
| xl | 32px | Vertical gap between major card sections |
| 2xl | 48px | Dashboard-level gap when card is used in stacked view |
| 3xl | 64px | Reserved for page-level layout, not in-card content |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--primary-background-color)` | Dashboard/background surface around card |
| Secondary (30%) | `var(--card-background-color)` | Card container, condition rows, summary tiles |
| Accent (10%) | `var(--state-icon-active-color)` | GO chip, active/primary window marker, launch-time highlight |
| Destructive | `var(--error-color)` | Explicit failure/error chips and destructive confirmations only |

Accent reserved for: GO status chip in window tiles, active best-window indicator, and launch-time emphasis in timing row. Do not use accent for neutral labels or non-critical metric values.
